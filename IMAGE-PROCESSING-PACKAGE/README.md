# Image_processing

Description. 
The package image_processing is used to:
	Processing :
		- Histrogram matching
		- Structual similarity
		- Resize image
	Utils:
		- Read image
		- Save image
		- Plot image
		- Plot result
		- Plot hitogram


## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install image_processing
	Processing 
	 
```bash
pip install image_processing
	Processing 
	```
